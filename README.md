#  T-Library: Console Management System

**T-Library** — это отказоустойчивое консольное приложение на языке **C#**, разработанное для эффективного управления личной библиотекой. Проект демонстрирует навыки объектно-ориентированного программирования (ООП), работы с файловой системой и современной сериализации данных.

##  Основной функционал

* **Полный цикл CRUD:** Добавление, просмотр, поиск и удаление книг в реальном времени.
* **Управление статусами:** Возможность отмечать книги как прочитанные или добавлять их в «Избранное».
* **Умный поиск:** Реализован регистронезависимый поиск по ключевым словам в названии, авторе или описании.
* **Персистентность данных:** Автоматическое сохранение и загрузка библиотеки из JSON-файла.
* **Гибкая настройка БД:** При запуске пользователь может выбрать имя существующего файла базы данных или создать новый.

##  Архитектурные особенности

Проект реализован с соблюдением принципа **Separation of Concerns** (Разделение ответственности):

1.  **Модель (`Book.cs`):** Чистый POCO-класс (Plain Old CLR Object) для хранения состояния сущности.
2.  **Ядро (`LibraryManager.cs`):** Бизнес-логика, использующая LINQ для фильтрации и `System.Text.Json` для работы с хранилищем.
3.  **Интерфейс (`Program.cs`):** Отказоустойчивый Command-loop с глубокой валидацией пользовательского ввода.
